# Movie Explorer App

A responsive React app that allows users to explore trending movies, search titles, and manage favorites.

## Features
- Login / Register with localStorage
- Theme toggle (Light/Dark Mode)
- View trending movies
- Search movies
- Favorite management
- Protected routes

##  Tech Stack
- React + Vite
- Tailwind CSS + MUI
- React Router
- TMDb API
